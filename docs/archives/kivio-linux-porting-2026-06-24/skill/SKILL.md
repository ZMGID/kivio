# Kivio Linux Porting Handoff

Use this archive as historical evidence only. For active work, read the current
`docs/progress/MASTER.md` first.

## Current Linux Baseline

- Target: Ubuntu 22.04.5 LTS, kernel 6.8.0-40-generic, X11.
- Bundle: AppImage generated by `npm run build` through `tauri.linux.conf.json`.
- Key runtime gate: packaged `KIVIO_DESKTOP_SMOKE=1` AppImage smoke.

## Main Boundaries

- `capture_port.rs` owns platform capture behavior.
- `ocr_port.rs` owns OCR availability and local OCR dispatch.
- `platform_capabilities.rs` owns UI-facing desktop capability status.
- `skills/discover.rs` owns AppDir skill discovery for `kivio-code`.
